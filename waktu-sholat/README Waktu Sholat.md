# Deskripsi 
Aplikasi ini menghitung waktu-waktu salat harian dan bulanan menggunakan pendekatan fisika astronomi. Setiap waktu shalat seperti Subuh, Terbit, Dzuhur, Ashar, Maghrib, dan Isya, dihitung berdasarkan posisi Matahari terhadap bumi, koreksi lintang-bujur, serta perbedaan zona waktu.
Aplikasi ini menyediakan antarmuka sederhana bagi pengguna untuk memilih:
- Kota
- Tanggal
- Bulan (Masehi atau Hijriyah)
- Tahun
Tujuannya adalah membantu pengguna mengetahui waktu shalat yang akurat sesuai lokasi dan tanggal yang diinginkan.
# Fitur Utama
- Perhitungan waktu sholat berbasis astronomi (bukan tabel statis).
- Pilihan kalender Masehi atau Hijriyah.
- Pemilihan kota secara fleksibel.
- Hasil waktu shalat ditampilkan secara lengkap untuk 1 hari dan 1 bulan.
# Teknologi
- Python dan GUI
- Fisika astronomi
- Konversi waktu matahari ke waktu lokal
- Perhitungan sudut deklinasi Matahari
